---
title: Dependencies
---

1. Clone the [lecca-io](https://github.com/lecca-digital/lecca-io.git) repo.
2. `cd lecca-io`
3. Install Node.js (version > 20) and npm
4. Install pnpm globally:
   `npm install -g pnpm`
5. Install dependencies:
   `pnpm install`
6. (Optional) Install NX Console VS Code extension.
