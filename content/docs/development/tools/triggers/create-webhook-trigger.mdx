<Step>
### Create new folder

Locate apps directory `/apps/server/apps/public` and add a new folder with your app name. E.g. `your-new-app`.

<Files>
  <Folder name="apps" defaultOpen>
    <Folder name="ui" />
    <Folder name="server" defaultOpen>
      <Folder name="src" defaultOpen>
        <Folder name="apps" defaultOpen>
          <Folder name="public" defaultOpen>
            <Folder name="your-new-app" />
          </Folder>
        </Folder>
      </Folder>
    </Folder>
  </Folder>
</Files>

</Step>

<Step>
### Create app file

Create a new file within your new app folder. Call it `your-new-app.app.ts`. Make sure it ends with `.app.ts`.

Add your own values. We provided mock data to help initialize the process.

1. Name the class: export class `YourNewApp`
1. Add an id: `your-new-app`
1. Add a name: `Your New App`
1. Add a description: `Does something cool`
1. Needs Connection: `false` if it doesn't need authentication, just runs a function.
1. Is Published: Set to `true` once it's ready to view in the UI.

```ts
import { Action } from "@/apps/lib/action";
import { App, AppContructorArgs } from "@/apps/lib/app";
import { Connection } from "@/apps/lib/connection";
import { Trigger } from "@/apps/lib/trigger";
import { ServerConfig } from "@/config/server.config";

export class YourNewApp extends App {
  id = "";
  name = "";
  logoUrl = `${ServerConfig.INTEGRATION_ICON_BASE_URL}/apps/${this.id}.svg`;
  description = "";
  needsConnection: false;
  isPublished = true;

  connections(): Connection[] {
    return [];
  }

  actions(): Action[] {
    return [];
  }

  triggers(): Trigger[] {
    return [];
  }
}
```

</Step>

<Step>
### Add new app to the apps object

Add your new app to the apps object found in the same directory you created your new app folder.
`/apps/server/apps/public/apps.ts`.

<Files>
  <Folder name="apps" defaultOpen>
    <Folder name="ui" />
    <Folder name="server" defaultOpen>
      <Folder name="src" defaultOpen>
        <Folder name="apps" defaultOpen>
          <Folder name="public" defaultOpen>
            <Folder name="your-new-app" defaultOpen>
              <File name="your-new-app.app.ts" />
            </Folder>
            <File name="apps.ts" />
          </Folder>
        </Folder>
      </Folder>
    </Folder>
  </Folder>
</Files>

Add it in alphabetical order to make development cleaner.

```ts
const _APPS = {
    ai: AI,
    anthropic: Anthropic,
    ...,
    ...,
    ...,
    ...,
    'x': X,
    'your-new-app': YourNewApp, // [!code highlight]
    'youtube': Youtube,
    'zoho-books': ZohoBooks,
    'zoho-crm': ZohoCrm
}
```
