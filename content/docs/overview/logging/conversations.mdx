---
title: Conversations
---

When messaging an AI agent a conversation is started. You can view all conversations on the conversations page found on the left navigation menu.

You will find conversations started via the chat UI, agents messaging other agents, and workflows messaging agents.

<Callout>
  `Conversations` are used on the frontend. But the server calls them `tasks`.
</Callout>

<Callout type="warn">
  All conversations are deleted between 3 and 30 days depending on the plan you
  are on. If you are self hosted, we are working on making these numbers
  configurable.
</Callout>
